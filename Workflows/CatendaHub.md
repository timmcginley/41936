# Catenda based workflow
This guide will help you find how to start using Catenda

## Why Use Catenda? 
Catenda helps you: 
* **Manage issues in a common data environment** – Create, assign, and track model-based issues (BIM collaboration format (BCF)) directly linked to the model, ensuring clear communication across teams.
* **Access models from any device** – View and navigate large BIM models on desktop, tablet, or mobile through a standard web browser without needing expensive software licenses.
* **Connect models and data** – Link federated BIM models with project documents, photos, and specifications in a single, organized platform.
* **Control access with open standards** – Manage user permissions and share project data securely using IFC and BCF, ensuring vendor-neutral collaboration.
* **Perform clash detection online** – Run automated clash detection tests on combined models directly in the cloud to identify coordination issues early.
* **Keep everyone aligned with activity streams** – Follow real-time project updates, comments, and version changes to maintain a single source of truth for the entire project team.
 
All inside your DTU Workspace at https://hub.catenda.com/signin
